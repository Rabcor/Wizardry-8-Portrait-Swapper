# Wizardry 8 Portrait Swapper
Now you can swap the portraits in Wizardry 8 with less pain.

# Installation
Download the executable from the release section and just run it.

You can also download the source from the src directory and run main.py directly with python.


<details>
  <summary>Info for running on python</summary>
  
- Python Version: 3.13
- Dependencies:
  - pillow
  - numpy
  - tkinterdnd2
  - struct
- Optional Dependencies:
  - libimagequant


</details>
